# PlanA Mobile

React Native (Expo) mobile app — **Phase 2 scope**.

This directory is a placeholder. Mobile development starts after the web app reaches a validated MVP with the Phase 1 test team.

## Planned stack

- React Native + Expo SDK 52+
- Expo Router (file-based navigation)
- Electric SQL React Native client (offline-first sync)
- NativeWind (Tailwind CSS for React Native)

## Getting started (Phase 2)

```bash
pnpm --filter @projecta/mobile start   # Expo dev server
```

iOS and Android builds will use EAS Build.
